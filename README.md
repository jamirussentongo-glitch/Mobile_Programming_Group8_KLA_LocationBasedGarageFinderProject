## Feature Set
- User registration and login
- Browse nearby garages
- Search garages by name/service
- View garage details
- Book a garage service
- View and manage bookings

## Technical Stack
- Kotlin
- Jetpack Compose (Material 3)
- Navigation Compose
- Hilt (DI)
- Room (local persistence)
- Retrofit (API layer)
- Coil (image loading)
- Google Maps / Location services
